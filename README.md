# repair_h5

> A Vue.js 移动端 维修管理

## Build Setup

``` bash
# 安装依赖包
npm install

#服务热加载  自动监听8080端口
npm run dev

# 项目打包压缩
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
#项目采用vue框架  借助vue-cli脚手架 搭建项目
#技术栈有 vue2.0+vue-router+axios(vue-resource的替代)+sass+webpack+mint-ui(UI框架)
#采用 lib-flexible 适配终端
